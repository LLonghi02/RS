Dear students,

For the optional coding project of the Recommender Systems course we can assign you the following papers:
- Walker et al. - 2022 - Recommendation via Collaborative Diffusion Generative Model (CODIGEM), https://github.com/WorldChanger01/CODIGEM
- Wu et al. - 2021 - Self-supervised Graph Learning for Recommendation (SGL), https://github.com/wujcan/SGL

Attached you will find:
- The papers
- The latex source of the technical report as well as the project requirements and some documentation
- The clone of the github repository to use, including examples

The goal is to port the methods in the framework attached, with the correct interface, in such a way as to obtain results that are as close as possible to the original ones. Ideally they should be within 1%.

The final deadline to deliver the projects is the end of march. Early delivery is possible especially if you want to register your grade in January or February.  There will be a checkpoint at the end of February where we will ask you to show that you have at least begun the porting process and confirmed that the original source code is executable and has all the necessary components. If you encounter significant issues, are unsure how to proceed or have specific questions on the project do not hesitate to contact us, it is best to address those rapidly. An example of issues encountered in the past are that the original implementation is missing important parts or that the results it provides are clearly anomalous.

Please confirm that you accept this project. 
Note that these projects are related to current research activities and therefore you should accept the assignment only if you are confident and motivated. Furthermore, the source code you deliver will be thoroughly checked and if major issues are found you will be required to correct them. 
I am sure it will not happen, but it is best to clarify that if the project is poorly done or the reporting is dishonest you will receive a failing grade.
